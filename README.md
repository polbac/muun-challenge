
# IP Blocklist Service Assignment

## API Doc

```
http://localhost:3000/api
```

## Commands

### Start the app (dev mode)

```bash
npm run start:dev
```

### Create a JWT token

```bash
npm run auth:token
```

### Run migrations

```bash
npm run migration:run
``` 

### Ingest blocked IPs

```bash
npm run ingest:ips
```
